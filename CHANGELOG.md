## [1.0.0] - 2026-03-02

### Added

- Initial release of Gittodoc skill for AI coding assistants and Claude Desktop
- Quick-start one-liner for generating and fetching repository digests
- Step-by-step curl workflow: trigger generation, extract S3 URL, download text
- macOS compatibility notes (grep -oE vs -oP)
- Output format documentation (summary, directory tree, file contents, token estimate)
- Working with output: save to file, search within digest, extract single files, check token count
- URL pattern reference
- Rate limits and caveats documentation
- Tips for effective usage
- Installation instructions for Cursor, Claude Code (personal and project), and Claude Desktop
- README with "Why a skill and not MCP?" rationale
